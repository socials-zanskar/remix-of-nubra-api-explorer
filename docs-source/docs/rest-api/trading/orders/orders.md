# Orders
The Orders API is a interface designed to facilitate seamless interaction with our trading platform's order management system. It allows you to create, modify, cancel, and track orders, supporting a variety of execution strategies to cater to diverse trading needs.

## Notes:
> As per the exchange mandate (NSE/INVG/67858), the current order rate limit is capped at 10 OPS / client. To place orders beyond this threshold, please reach out to our support team at support@nubra.io to initiate the process of registering your algorithm with the exchange.
